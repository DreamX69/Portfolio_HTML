การเขียนสะท้อนตนเองแบบการเรียนรู้ผ่านประสบการณ์ Week # 1
(Recursive) ประสบการณ์ในสัปดาห์นี้มีการเชื่อมต่อกับประสบการณ์หรือแผนการที่ผ่านมาหรือไม่? มี เพราะ ตอนฝึกงานหลักๆที่ทำเป็นประจำเป็นการเขียนเว็บ จะได้นำความรู้ที่มีมาใช้ในวิชานี้
(CE) อะไรคือประสบการณ์ที่ผ่านมาของคุณ ? ได้ฝึกงานที่บริษัท JONN Power Idea ภาษาที่ได้เขียหลักๆจะมี Python React Xml JavaScript
(RO) อะไรคือประสบการณ์ที่สำเร็จและล้มเหลว ? ประสบการณ์ที่ล้มเหลว คือ เวลาคุยงานกับชาวต่างชาติสื่อสารได้ไม่ลื่นไหล ทำให้ต้องคุยกันนานถึงจะเข้าใจกันได้ ประสบการณ์ที่สำเร็จ คือ พอฝึกงานเสร็จที่บริษัทก็จ้างทำงานต่อเป็น Out Source
(AC) คุณได้เรียนรู้อะไรจากประสบการณ์ ? ได้เรียนรู้ข้อผิดพลาดของตัวเอง ว่ายังขาดความรู้ส่วนไหนไปบ้าง แล้วใช้เวลาว่างเติมเต็มความรู้ในส่วนที่ยังขาด
(AE) คุณจะได้นำสิ่งที่เรียนรู้จากประสบการณ์ ไปใช้อย่างไรต่อไป ? ฝึกทักษะภาษาอังกฤษและทักษะการเขียนโปรแกรมควบคู่กันไปอย่างสม่ำเสมอ
